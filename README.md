# Cyber Glass Term

Cyber Glass Term is a lightweight and minimal terminal emulator for Linux built using C and GTK/VTE. It focuses on performance, simplicity, and a modern glass-like UI experience.

---

## 🚀 Features
- Lightweight and fast terminal
- GTK + VTE based UI
- Custom window behavior support
- Minimal resource usage
- Linux focused design

---

## 📦 Requirements
Make sure you have these installed:

- gcc
- make
- gtk+-3.0
- vte-2.91

On Arch Linux:
```bash
sudo pacman -S gtk3 vte3 base-devel
